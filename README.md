# construct_sisifo_software
Este repositorio contiene una serie de script para dockerizar el proyecto SISIFO UD



### Correr
```bash
# Descarga los repositorios necesarios
python2.7 get_repos.py
```
```
# correr odoo
docker-compose up
```
